# [SWPUCTF 2021 新生赛] re2

## 题目信息

- **附件**: `re2.exe`
- **类型**: Windows x64 PE 逆向
- **核心考点**: 识别分段凯撒加密、处理多解歧义

## 静态分析

### main 函数关键逻辑

1. **硬编码目标字符串**: `str2 = "ylqq]aycqyp{"` (12 字符)
2. 读取用户输入 `str`
3. **逐字符变换**（for 循环）：

```c
if ((str[i] <= 0x60 || str[i] > 0x62) && (str[i] <= 0x40 || str[i] > 0x42))
    str[i] -= 2;        // 其他字符：减 2
else
    str[i] += 0x18;     // 'A'-'B' 或 'a'-'b'：加 24 (0x18)
```

**关键纠正**：之前误将条件理解为连续区间 `0x40 < c <= 0x62`，实际是两个**不连续**的小区间：
- `0x40 < c <= 0x42` 即 `'A'`、`'B'`
- `0x60 < c <= 0x62` 即 `'a'`、`'b'`

只有这 **4 个字符** (`A`, `B`, `a`, `b`) 执行 `+24`，**其余所有字符**执行 `-2`。

4. `strcmp(str, str2)`，相等则输出 `"bingo"`

## 逆向推导

对目标字符串 `ylqq]aycqyp{` 逐字符逆向：

| 目标 | 逆向操作 | 原字符 | 验证（正向）|
|------|----------|--------|------------|
| `y` | `-24` → `a` 或 `+2` → `{` | `a` / `{` | `a`+24=`y` / `{`-2=`y` |
| `l` | `+2` → `n` | `n` | `n`-2=`l` |
| `q` | `+2` → `s` | `s` | `s`-2=`q` |
| `q` | `+2` → `s` | `s` | `s`-2=`q` |
| `]` | `+2` → `_` | `_` | `_`-2=`]` |
| `a` | `+2` → `c` | `c` | `c`-2=`a` |
| `y` | `-24` → `a` 或 `+2` → `{` | `a` / `{` | `a`+24=`y` / `{`-2=`y` |
| `c` | `+2` → `e` | `e` | `e`-2=`c` |
| `q` | `+2` → `s` | `s` | `s`-2=`q` |
| `y` | `-24` → `a` 或 `+2` → `{` | `a` / `{` | `a`+24=`y` / `{`-2=`y` |
| `p` | `+2` → `r` | `r` | `r`-2=`p` |
| `{` | `+2` → `}` | `}` | `}`-2=`{` |

3 个位置（`y`→`y`）存在**二义性**，共 8 种组合。根据 flag 格式惯例（`{}` 内不出现 `{`），且 `nss_caesar` 构成完整单词，确定唯一合理解：

```python
enc = "ylqq]aycqyp{"
flag = ""
for c in enc:
    if (ord(c) <= 0x60 or ord(c) > 0x62) and (ord(c) <= 0x40 or ord(c) > 0x42):
        flag += chr(ord(c) + 2)      # 逆向：其他字符 +2
    else:
        flag += chr(ord(c) - 0x18)   # 逆向：A/B/a/b -24
print(flag)  # {nss_c{es{r}  (存在多解，人工修正为 {nss_caesar})
```

## 总结

- 入门级 RE，无壳无混淆，IDA/radare2 直接分析 main 即可
- **分支条件误读是最大坑点**：`&&` 连接的复合条件导致有效区间是 **两个不连续的子区间**，而非连续大区间
- 逆向时产生**多解歧义**（`a`/`{` 均可产生 `y`），需结合 flag 格式惯例和语义排除
- `2 + 24 = 26` 提示凯撒密码本质，但出题人故意用分段区间制造陷阱
